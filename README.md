# Intro PHP

## Afficher les erreurs avec MAMP
Applications/MAMP/bin/php/PHP VERSION/conf/php.ini
(Remplacez PHP VERSION par votre version de PHP)

Cherchez `display_errors = Off`
Remplacez par `display_errors = On`
